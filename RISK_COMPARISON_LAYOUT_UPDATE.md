# Risk Analysis 对比图布局更新

## 更新内容

修改了 `risk_analysis_batch.py` 中的对比图生成逻辑，采用固定的 2×2 布局。

## 修改前

- 自动布局：1-3个场景单行，更多场景网格布局
- 热力图尺寸会被压缩
- 子图大小不固定

## 修改后

- **固定 2×2 布局**：每个热力图保持原始尺寸（14×10）
- **不压缩尺寸**：确保热力图清晰可读
- **最多 4 个场景**：一次对比最多显示 4 个场景
- **统一颜色条**：右侧显示，便于对比

## 布局示例

```
┌─────────────────────────────────────────────────────┐
│  [Scenario 1]        [Scenario 2]        [Colorbar]│
│  (14×10)             (14×10)             0.0 - Max  │
│                                                      │
│  [Scenario 3]        [Scenario 4]                   │
│  (14×10)             (14×10)                        │
└─────────────────────────────────────────────────────┘
```

## 技术细节

```python
# 固定使用 2x2 布局
n_cols = 2
n_rows = 2

# 每个子图保持原始尺寸
fig_width = 14 * n_cols   # 每个子图宽度 14
fig_height = 10 * n_rows  # 每个子图高度 10
```

## 使用场景

- **多时段对比**：白天/夜间、旱季/雨季
- **多场景分析**：不同配置或假设
- **批量报告**：自动生成对比图

## 注意事项

- 一次最多显示 4 个场景
- 超过 4 个场景时，建议分批处理或生成多张对比图
- 每个热力图保持原始尺寸，确保清晰度
